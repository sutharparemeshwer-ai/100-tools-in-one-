Custom Accordion

Vanilla JS & CSS implementation of a collapsible FAQ.

How does this custom accordion work? +

It works by using JavaScript to toggle a CSS class (like .active) on the header when clicked. The CSS handles the smooth transition of the content's height, making it appear to expand or collapse gracefully.

Why use Vanilla JavaScript instead of a library? +

Building components from scratch helps reduce the overall size of the project (less external code) and demonstrates a deeper understanding of fundamental web technologies. It gives you full control over the component's behavior and styling.

Can I have multiple accordion items open at once? +

Yes! Unlike some library implementations, this Vanilla JS code is designed to allow multiple items to be open simultaneously. You can easily modify the JavaScript logic if you prefer to close other items when a new one is opened (known as "single-open" mode).
